# Quick Start Guide for aix on macOS

## 1. Install aix using uv

First, install uv if you don't have it:
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then install aix:
```bash
uv tool install aix --from git+https://github.com/bhadzhiev/AIeXporter.git
```

## 2. Set up API provider

**Option A: OpenRouter (free tier available)**
```bash
# Get API key from https://openrouter.ai
aix api-key openrouter
aix config --set default_provider=openrouter
aix config --set custom_providers.openrouter.default_model="microsoft/mai-ds-r1:free"
```

**Option B: OpenAI (paid but reliable)**  
```bash
# Get API key from https://platform.openai.com
aix api-key openai
aix config --set default_provider=openai
```

**Option C: Anthropic (paid, high quality)**
```bash
# Get API key from https://console.anthropic.com
aix api-key anthropic  
aix config --set default_provider=anthropic
```

## 3. Import collections from GitHub

Add pre-built prompt collections:
```bash
aix collection-import-repo https://github.com/bhadzhiev/AIeXpoerterCollections.git
```

## 4. Run project analysis

Navigate to any project directory and run:
```bash
aix run project-info
```

**Pro tip:** Use `aix list` to see all available prompts from the imported collections.

**Note:** The `microsoft/mai-ds-r1:free` model is available on OpenRouter's free tier. For more models or guaranteed availability, consider OpenAI or Anthropic with credits.
